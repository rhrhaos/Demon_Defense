## 🎮 플레이 방법

1. `index.html` 파일을 브라우저에서 열기 (별도 설치 불필요!)
2. 에디터에서 맵 편집 (벽, 입구, 마왕성 배치)
3. 상점에서 스포너/함정 구매 → 맵에 배치
4. ▶️ 전투 시작 → 마왕 스킬로 용사 저지
5. 스테이지 클리어 → 마왕 강화 선택 → 반복!

## ✨ 주요 특징

### 🗺 맵 에디터
- 자유로운 맵 디자인 (벽, 바닥, 입구, 마왕성)
- Undo/Redo 지원 (`Ctrl+Z` / `Ctrl+X`)
- 드래그로 빠른 벽 배치

### 🐉 6개 몬스터 패밀리 (18종 + 엘리트 3종 + 하이브리드 3종)
| 패밀리 | 기본 → 진화1 → 진화2 |
|--------|----------------------|
| ⚔ 근접 | 근접병 → 스킬전사 → 돌진병 |
| 🏹 원거리 | 궁수 → 관통궁수 → 저격수 |
| 💚 힐러 | 힐러 → 강화힐 → 정화힐 |
| 💣 폭탄 | 폭탄병 → 분열폭탄 → 얼음폭탄 |
| ⏳ 시간 | 시간술사 → 시간버프 → 시간정지 |
| 🟢 슬라임 | 슬라임 → 분열슬라임 → 킹슬라임 |

### 🧬 스포너 합성
- ⚔ + 🏹 = **검궁사** (근접+원거리 하이브리드)
- 💚 + 💣 = **독폭탄** (독장판+폭발)
- ⏳ + 🟢 = **시간슬라임** (감속+분열)

### 🔮 마왕 실시간 스킬 (7종)
Meteor · Frost Nova · Curse · Wall · Fear · Overclock · Buff

### 👑 마왕 성장 시스템 (30개 패시브)
매 스테이지 후 **6개 카테고리**에서 업그레이드 선택:

| 카테고리 | 예시 |
|---------|------|
| 🗡 전투 | 흡혈, 광전사, 독무기, 크리티컬 |
| 🧬 변이 | 사망분열, 거대엘리트, 부활, 재생 |
| ⚙ 메커닉 | 이중소환, 함정강화, 대군세 |
| 🔮 스킬 | 에너지증폭, 쿨감, 혜성폭격 |
| 💰 경제 | 이자, 할인, 현상금 |
| 💀 저주 | 유리대포, 불안정, 혼돈의축복 |

3단계 티어: 일반 → 희귀 → 전설

### 🛡 용사 적응 시스템
용사는 사망 원인을 학습하여 대응 능력을 획득:
- ☠ 독 → 독 면역 + 자동 정화
- ⚔ 근접 → 가시 반격 + HP 증가
- 🔫 탄환 → 회피 + 반사
- 👹 보스 → 추가 피해 + CC 저항
- ...등 6종

### 🦸 용사 4클래스 (스킬 4개씩)
| 클래스 | 스킬 |
|--------|------|
| ⚔ 전사 | 도발 · 강타 · 돌진 · 전투함성 |
| 💚 힐러 | 치유 · 정화 · 방어막 · 부활 |
| 🏹 레인저 | 저격 · 화살비 · 독화살 |
| 🔮 마법사 | 파이어볼 · 체인라이트닝 · 텔레포트 |

### 📊 시너지 시스템
같은 패밀리 스포너를 여러 개 배치하면 시너지 보너스 발동!

## 🛠 기술 스택

- **순수 JavaScript** — 프레임워크/라이브러리 없음
- **단일 HTML 파일** — 서버 불필요, 브라우저만 있으면 OK
- **Canvas 2D** 렌더링
- **BFS 경로탐색** (용사/몹 AI)
- **약 2,600줄**의 코드
