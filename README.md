## 🎮 플레이

**온라인:** GitHub Pages로 배포하면 바로 플레이 가능!

**로컬:** `index.html`을 브라우저에서 열기 (assets 폴더 필요)

## 📁 프로젝트 구조

```
├── index.html              ← 게임 본체 (~150KB)
├── README.md
└── assets/
    ├── mobs/               ← 몬스터 24종
    ├── heroes/             ← 용사 4종
    ├── spawners/           ← 스포너 21종
    └── bullets/            ← 탄환 3종
```

## ✨ 주요 특징

- 🗺 **맵 에디터** — 자유로운 던전 설계
- 🐉 **6개 몬스터 패밀리** (24종) + 스포너 합성
- 🔮 **마왕 실시간 스킬** 7종 (메테오, 저주, 공포 등)
- 👑 **마왕 성장 시스템** — 30개 패시브 (6카테고리, 3티어)
- 🛡 **용사 적응 AI** — 사망 원인을 학습하여 대응
- 🎨 **픽셀아트 리소스** — 커스텀 스프라이트

## 🚀 GitHub Pages 배포

1. Settings → Pages
2. Source: Deploy from branch → `main` / `/ (root)`
3. Save → 1~2분 후 `https://아이디.github.io/저장소명/` 에서 플레이!
